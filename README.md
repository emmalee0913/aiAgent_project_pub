# 📸 Mission Judge Agent  
### (Gemini + LangChain + Streamlit)

부모가 입력한 **미션 세부사항**과 자녀가 제출한 사진 증거(로컬 경로)를 바탕으로  
미션의 완수율(%)과 **통과/반려 여부**를 자동으로 판단하는 Streamlit 기반 AI 에이전트입니다.

---

## ✨ 주요 기능
  1. 미션 입력 → 즉시 미션 요약 & 체크리스트 생성
  2. 사진 경로를 하나씩 추가하며 관리 (최대 10장)
  3. 사진 관찰 요약 / 전후 변화 / 한계 분석
  4. 최종 판정 (완수율 %, 통과/반려)
